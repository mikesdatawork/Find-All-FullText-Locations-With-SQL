## About-Process

<p>Here's some quick SQL logic to give you the location of your FullText Catalogs. Additionally; it will show you which FullText is Offline, or Online.</p>      


## SQL-Logic
```SQL
use master;
set nocount on
select
'database' = db_name(database_id)
, 'state desc' = state_desc
, 'type desc' = type_desc
, 'path' = physical_name
from
sys.master_files where file_id &amp;amp;amp;gt; 2 and type &amp;amp;amp;gt; 1
order by
db_name(database_id) asc

```
